優良企業が参加する選考直結イベントが開催！

今回ご紹介するのは、優良企業の早期内定を狙える選考直結イベント『JobTryout』です。

当日はグループワークに挑戦し、各企業の人事担当者が一人ひとり丁寧に評価します。
各企業の人事担当者から高い評価を得ると、このイベント独自の特別選考ルートに進むことが可能です。優良企業の早期内定を狙えるチャンスです！

このイベントに参加するメリット

メリット1：ここでしか出会えない優良企業が参加

ここ数年の就活では、多くの企業が正式スケジュールよりも前倒しで動く傾向が強まっています。3月の情報解禁前にどれだけ準備を進められたかで、結果が大きく左右されるのです。
今の時期、ここでしか出会えない大手 / 優良企業にアピールして、自分の就活を有利に進めましょう！

参加実績
三井物産、伊藤忠グループ、楽天株式会社、エイベックス・グループ・ホールディングス株式会社、ユニ・チャーム株式会社、帝人株式会社、KDDIグループ、住友重機械工業株式会社（住友グループ）、ロート製薬株式会社、株式会社サイバーエージェント

ほかにも、各業界を代表する企業の参加実績がいくつもあります。

メリット2：選考直結型だから早期内定のチャンスあり！

当日優秀な成績を残した学生は、1次選考スキップ権・人事部長面接権・役員面接権など、特別選考ルートへ進む権利を獲得できます。だから早期内定のチャンスが大きい！

メリット3：当日の座談会では、人事・経営者らからアドバイスを受けられる

イベント当日の座談会では、人事・経営者らと直接会話をすることができます。 業界や仕事について深く知ることが可能で、さらに自己分析のやり方からES・面接のアドバイスまで、内定に必要なアドバイスを受けられることも！

当日の内容（イベント種別）

  • 日程毎にイベント内容が異なりますのでご注意ください。
  • 業界体感プログラム：INDUSTRY
    日本経済30年を舞台に、チームで企業経営を行うビジネスゲームです。
    日本を代表する14業界の業界研究を行うことができます。
  • 業務改善プログラム：IMPROVEMENT
    山ほどある物流配送案件を戦略的に解決するビジネスゲームです。
    高い戦略性、段取り力、チームワーク、営業力、行動力が問われます。
    仕事の本質、仕事の効率的に改善する方法が学ぶことができます。
  • 組織体感プログラム：DISCOVERY
    チームで協力して仕入・製造・販売を行うビジネスゲームです。
    制約条件が厳しく、ビジネスに体感することできます。
    様々な職種の役割・やりがいを学ぶことができます。
  • 職種体感プログラム：REALIZE
    「経営企画」「営業」「システムエンジニア」の3職種を体感するビジネスゲームです。
    それぞれの役割・やりがいを理解することができます。
  • 経営戦略理解プログラム：INNOVATION
    日本経済30年を舞台にメーカーを経営します。
    1990年代から現在に至る市場環境の変化を経営者の立場で体感することができます。
  • 顧客課題解決プログラム：SOLUTION SALES
    クライアント側とソリューションセールス側に分かれて、クライアントの課題をヒアリングし、提案を行います。
    実際の仕事シーンに一番近いビジネスゲームです。
  • 上記6つのプログラムがあり、それぞれ1回ずつ最大6回参加可能です。
